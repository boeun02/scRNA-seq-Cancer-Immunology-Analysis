# Multimodal Tumor Microenvironment Modeling

## Overview
This project aims to model cell-cell interactions in tumor microenvironments
using spatial transcriptomics and gene expression data.

## Goals
- Analyze spatial proximity vs gene expression similarity
- Build graph-based cell interaction models
- Visualize tumor microenvironment structure

## Tech Stack
- Python (Scanpy, PyTorch)
- GNN (PyTorch Geometric)
- FastAPI + React
